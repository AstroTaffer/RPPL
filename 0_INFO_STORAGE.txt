Приоритет задания параметра:
	1. kwargs
	2. Файл конфига
	3. Значение по умолчанию

for parameter in class.pars:
	if [in kwargs]
	elif in [config]
	else [default]
В случае ссылки на другие параметры при задании новых, будет работать начиная с той версии питона, когда словари сделали упорядоченными
